# Meus Sites Favoritos

Projeto desenvolvido em HTML para a atividade da escola.

## Sobre o projeto

A página mostra 5 sites favoritos com:

- Nome do site
- Imagem
- Link para acesso

- GitHub
- Spotify
- Netflix
- Google
